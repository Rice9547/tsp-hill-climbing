# Solve TSP problem by Hillclimbing
## 環境
Ubuntu 18.04.2 LTS
## 使用
```
git clone https://github.com/Rice9547/tsp-hill-climbing.git && cd tsp-hill-climbing
sudo ./install.sh
python3 solve.py
```
Get total distance in terminal, and result image in result.png!
